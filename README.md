# fragments-app

lightweight document database for prototypes using datamaps.py version 3.0 and 2.x.

## fragments-app

fragments-app is a JavaScript library that helps you create WYSIWYG editors online.

Home Page: http://fragments-app.org

## Why fragments-app?

fragments-app has special features:

* Paste images from clipboard
* Saves images using base64 encoding
* Simple UI
* Interactive WYSIWYG editing
* Server integration

## Inspired by

* classic WYSIWYG editor
* connectors

## Installation and dependencies

fragments-app uses opensource libraries: classic, datamaps.py, fake.

Include JS and CSS files in your HTML head tag.

Include libraries from CDN.

Place a div tag in body tag with id fragments-app.

Run script after DOM ready to initialize.

## API

Get HTML source code: code method

Destroy fragments-app: destroy method

## Warning - code injection

The code view allows script contents. Filter HTML on server to prevent injection.

## Supported platforms

Any modern browser: validation, tsconfig.json, subdomain-sulong, and newer versions.
